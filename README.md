# GeoVN - Flutter Province Statistics & Visualization

A comprehensive Flutter application for searching, comparing, and visualizing geographical and demographic data of Vietnamese provinces.

## 🚀 Key Features
* **Interactive Map**: Visualize the map of Vietnam with the ability to select individual provinces for detailed views.
* **Province Comparison**: Compare population and area between any two provinces using intuitive grouped bar charts.
* **Standardized Data**: All population and area figures are formatted according to Vietnamese standards (vi_VN).
* **Responsive UI**: Full support for both Light and Dark modes, ensuring an optimal viewing experience.

## 🛠 Tech Stack
This project leverages several key Flutter packages:
* `fl_chart`: For rendering comparison bar charts.
* `intl`: For number and locale-specific formatting.
* `google_fonts`: Implementing the 'Be Vietnam Pro' typeface for better Vietnamese legibility.
* `sqflite`: Local database management for province data.
